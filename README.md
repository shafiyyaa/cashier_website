# Cashier Web - HTML, CSS, Bootstrap

## Introduction

Cashier Web is a simple, responsive website built using HTML, CSS, and Bootstrap. The website serves as a landing page for a cashier company website system, providing a clean and modern interface.

Live site: [Cashier Web](https://cashierwebhtmlcssbootstrap-jo4etyir0.vercel.app/)

## Features

- **Responsive Design**: Built with Bootstrap to ensure the site is fully responsive.
- **Animations**: Smooth animations to enhance user experience.
- **Carousel**: Interactive carousel component for showcasing items or promotions.
- **HTML, CSS, and Bootstrap Slicing from Figma**: The design is carefully converted from a Figma prototype to ensure consistency and precision.
  [link figma : https://www.figma.com/design/B6vCITu1dsxZhjNYfRlM8X/Cashier-Finance-Website-Design-(Community)?node-id=0-1&t=S89uAOt0LE6iBODI-1 ]

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
